# AR Pulse Dashboard V1

Amazon AR/WHD Performance Dashboard for Tampermonkey.

## Features

* AR/WHD performance visibility
* Quarter-based performance grid
* Gap tracking and review
* WHD summary cards
* Popup dashboard
* Full webpage dashboard
* Dark / Bright theme
* ASIN title loading
* FC Research integration
* Shift commentary
* Support hours visibility
* TPH tracking
* Item category classification
* Gap analysis between scans

---

# Installation

## 1. Install Tampermonkey

Download:

[https://www.tampermonkey.net/](https://www.tampermonkey.net/)

---

## 2. Install Script

Open the RAW script link:

```text
https://raw.githubusercontent.com/jenishdhl/AR-Performance-DB/main/AR_Pulse_Dashboard_V1.user.js
```

Tampermonkey will automatically detect the script.

Click:

```text
Install
```
